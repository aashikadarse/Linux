```bash
Farm/
	Coop/ 
		Chickens/
		Geese/
	Stable/
		Horses/
```

2. List the contents of the `Farm` directory.
3. "Move" into the `Coop` folder.
4. List the contents of the `Coop` folder.
5. "Move" into the `Chickens` folder.

9. How many geese (files) are in the `Geese` directory?

Farm Directory Exercise - Questions, Commands, and Output

1. Open a new terminal window. Navigate to the `Farm` folder.
Command:
cd Farm

2. List the contents of the `Farm` directory.
Command:
ls
Output:
Coop  Stable

3. Move into the `Coop` folder.
Command:
cd Coop

4. List the contents of the `Coop` folder.
Command:
ls
Output:
Chickens  Geese

5. Move into the `Chickens` folder.
Command:
cd Chickens

6. List out the chickens in the `Chickens` folder. How many are there?
Command:
ls
Output:
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Buckbeak 
-rw-r--r-- 1 aashika aashika 0 Feb 14 2071 Elvis 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Ethel 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Frida 
-rw-r--r-- 1 aashika aashika 0 Nov 12 2021 Hippo 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Jaba

7. Which chicken is very old? (Oldest modification time)
Command:
ls -l --sort=time
Output (oldest is last):
-rw-r--r-- 1 aashika aashika 0 Feb 14 2071 Elvis 
-rw-r--r-- 1 aashika aashika 0 Nov 12 2021 Hippo 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Buckbeak 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Ethel 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Frida 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Jaba
Elvis (future year)
8. In a SINGLE command, move from `Chickens` to `Geese`.
Command:
cd ../Geese

9. How many geese (files) are in the `Geese` directory?
Command:
ls
Output:
Amelia  Boris  Lucy  Muffin

10. Which goose is the largest? (largest file)
Command:
ls -l
Output:
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Amelia 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Boris 
-rw-r--r-- 1 aashika aashika 0 May 3 2021 Lucy 
-rw-r--r-- 1 aashika aashika 15 May 3 2021 Muffin
Muffin  (size 15)

11. Navigate to the `Horses` directory.
Commands:
cd ..
cd ../Stable
cd Horses 
   or
cd ../../Stable/Horses

12. How many horses are in the `Horses` directory?
Command:
ls
Output:
Archer  Buttons  Cookie  Jett

13. Hidden horse name (hidden file)?
Command:
ls -a
Output:
Archer  Buttons  Cookie  Jett Troy
